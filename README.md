# 🎓 Student Marks Prediction System

## 📌 Project Overview

This is a Machine Learning web application developed using Python, Scikit-learn, and Streamlit. The application predicts a student's final marks based on:

- 📚 Hours Studied
- ✅ Attendance Percentage
- 📝 Previous Marks

## 🚀 Features

- Machine Learning Prediction
- User-Friendly Streamlit Interface
- Real-time Prediction
- CSV Dataset Support
- Professional UI

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## 📂 Project Structure

Student-Marks-Prediction/

- app.py
- train_model.py
- student_marks.csv
- model.pkl
- requirements.txt
- README.md

## ▶️ How to Run

Install all libraries:

```bash
pip install -r requirements.txt
```

Train the model:

```bash
python train_model.py
```

Run the Streamlit application:

```bash
python -m streamlit run app.py
```
